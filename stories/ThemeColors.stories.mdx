# Theme colors

`FuriousThemeProvider` provides color palette for:

- primary, secondary
- success, info, warning, error
- foreground, background, gray scale

```ts
export interface ThemeColor {
  palette: {
    '10': string
    '20': string
    '30': string
    '40': string
    '50': string
    '60': string
    '70': string
    '80': string
    '90': string
    '100': string
  }
  light: string
  dark: string
  normal: string
}
```
